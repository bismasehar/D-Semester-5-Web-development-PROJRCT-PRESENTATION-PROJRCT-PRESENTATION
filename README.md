Modern Building Design

Description:
An academic project focused on designing modern building layouts with emphasis on both structure and aesthetics. The project demonstrates the application of architectural principles to create functional and visually appealing designs.

Features:

Developed detailed 3D models of building layouts.

Created comprehensive architectural plans for effective visualization.

Emphasized structural integrity while maintaining modern aesthetics.

Technologies Used:

Html,Css,Javascript,React,Asp.net

Purpose:
This project aims to combine theoretical knowledge of architecture with practical design skills, providing a clear representation of modern building concepts.
